*Programa "AproxSubsetSumm" basado en el algoritmo e-aproximación del libro T. Cormen*

+++Compilación:

Se compila el archivo con extensión java de la siguiente manera desde la carpeta contenedora del archivo AproxSubsetSumm.java
	javac AproxSubsetSumm.java

+++Ejecución:

Se ejecuta el archivo AproxSubsetSumm.class desde la carpeta contenedora
	java AproxSubsetSumm

Para usar el programa, pide al usuario introducir valores (n,[],t,epsilon)

+++Ejemplo de ejecución:

	Cantidad de valores del conjunto (ejemplo: 4):
	~/	4
	Ingresa los valores del conjunto (ejemplo: 104 102 201 101):
	~/	104 102 201 101
	Ingresa un valor epsilon (ejemplo: .4):
	~/	.4
	Ingrese el valor de t(ejemplo: 308):
	~/	201
	****************************
	El algoritmo se aplicará para
	     t=201
	     e=0.4
	En el conjunto
	     S=[104, 100, 101, 102]
	El resultado es:201
